# Scouting Window

The Scouting Window is the between-match shop. It is not part of the first slice, but the card schema and later run tables should prepare for it.

## Currency

- Currency name: Budget
- Base income after match: +5 Budget
- Interest: +1 per 10 saved Budget
- Interest cap: +3

## Shop Actions

- Buy card from board.
- Sell card.
- Reroll board for 2 Budget.
- Freeze board for free.
- Buy Scout XP for 4 Budget.

## Scout XP

Buying XP costs 4 Budget for +4 Scout XP.
